AI-powered SEO content planning and generation system for health, wellness, and lifestyle websites.

== Description ==

Doorillio Content Generator is a powerful WordPress plugin that helps you create high-quality, SEO-optimized content using AI technology. Perfect for health, wellness, and lifestyle websites, this plugin provides comprehensive content planning and generation tools.

= Key Features =

* **AI-Powered Content Generation** - Create comprehensive, SEO-optimized articles with just a few keywords
* **Research Integration** - Built-in topic research capabilities to enhance content quality
* **Multiple Content Formats** - Generate content with images, FAQs, infographics, and interactive charts
* **SEO Optimization** - Advanced SEO settings for maximum search engine visibility
* **Content Planning** - Strategic content calendar and planning tools
* **Website Analysis** - Analyze your existing content for optimization opportunities
* **CSV Import** - Bulk import content plans and topics
* **Customizable Tone** - Choose from multiple writing tones and styles
* **Target Audience Settings** - Tailor content for specific audience segments

= Content Types Supported =

* Meal Planning & Nutrition
* Health & Fitness
* Marketing & Business
* Online Income & Freelancing
* Technology & Software
* General Topics

= Advanced Features =

* Interactive content visualization
* Content clustering and topic organization
* Automated image suggestions with Unsplash integration
* Real-time content preview with multiple formats (Markdown, HTML, Preview)
* WordPress post creation integration
* Content statistics and reading time calculation

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/doorillio-content-generator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Content Generator menu item to access the plugin features

== Frequently Asked Questions ==

= Do I need an API key? =

The plugin includes built-in content generation capabilities. You can optionally configure external API keys for enhanced features in the settings.

= What types of content can I generate? =

The plugin specializes in health, wellness, and lifestyle content but can generate articles on any topic. It's particularly effective for meal planning, fitness, marketing, and online business content.

= Can I customize the generated content? =

Yes! The plugin offers extensive customization options including tone, length, SEO level, target audience, and content structure preferences.

= Does it include images? =

Yes, the plugin can suggest relevant images from Unsplash and generate interactive charts and infographics to enhance your content.
